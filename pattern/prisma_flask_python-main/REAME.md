# An application that works with Prisma and Flask

## Description
Api rest that works with Prisma and Flask

## Install environment

```shell    
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Install Prisma

```shell
pip install -U prisma

prisma db push
```

## Run application

```shell
flask run
```

or 

```shell
python app/main.py
```
